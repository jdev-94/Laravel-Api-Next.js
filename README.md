# 🚀 Sistema de Gestión de Productos - Proyecto Fullstack

Este repositorio contiene la solución completa para la gestión de productos, compuesta por un backend robusto en **Laravel** y un frontend moderno en **Next.js**.

## 🏗️ Arquitectura del Proyecto

El proyecto está dividido en dos aplicaciones independientes que se comunican a través de una API REST.



### 📂 Estructura del Repositorio

```text
├── api-crud/           # Backend: API RESTful en Laravel 12
└── crud-frontend/      # Frontend: Aplicación web en Next.js 14
